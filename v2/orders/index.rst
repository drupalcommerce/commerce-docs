Orders
======

Orders contain a list of order items and customer information. Orders
have states that are controlled through State Machine.

`Orders and order items <order-items.rst>`__ - Orders contain order
items, which represent purchased items.

`Understanding order types <order-types.rst>`__ - You can have different
order types. Order types have their own settings when it comes to cart,
checkout, and its processing.

`Order Processing <order-processing.rst>`__ - Allows you to process an
order, when the system recalculates order item prices and availability.
